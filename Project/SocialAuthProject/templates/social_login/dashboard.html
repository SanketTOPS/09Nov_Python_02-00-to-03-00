Developer Setup Guide

To test these social logins locally, you'll need to create OAuth applications on Google and GitHub and register them in the Django Admin. Follow the steps below:

Open your command line and run the following command to create an administrator account:

python manage.py createsuperuser

Follow the prompts to enter a username, email, and password. You will use these credentials to log in to the Django Admin panel.

  1. Visit the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Navigate to APIs & Services > OAuth consent screen. Configure it as External and fill in mandatory fields.
  4. Go to Credentials, click + Create Credentials, and select OAuth client ID.
  5. Choose Web application as the application type.
  6. Under Authorized redirect URIs, add: 
    http://127.0.0.1:8000/accounts/google/login/callback/
  7. Click Create and copy the generated Client ID and Client Secret.
  1. Navigate to your GitHub profile settings > Developer settings > OAuth Apps.
  2. Click the New OAuth App button.
  3. Fill in the fields:
    • Application Name: SocialAuth Local
    • Homepage URL: http://127.0.0.1:8000/
    • Authorization callback URL: 
      http://127.0.0.1:8000/accounts/github/login/callback/
  4. Click Register application.
  5. Copy the **Client ID** and click Generate a new client secret. Save the secret carefully.
  1. Start your server: 
    python manage.py runserver
  2. Navigate to http://127.0.0.1:8000/admin/ and log in with your superuser.
  3. Under Sites, select the existing site (usually `example.com`), modify its domain and display name to: 
    127.0.0.1:8000
    and click Save.
  4. Under Social accounts > Social applications, click Add social application:
    • Provider: Google (or GitHub)
    • Name: Google App (or GitHub App)
    • Client id: [Paste your Client ID]
    • Secret key: [Paste your Client Secret]
    • Sites: Select 127.0.0.1:8000 and click the arrow to move it to "Chosen sites"
  5. Click Save. You're fully configured! Logout and test!
